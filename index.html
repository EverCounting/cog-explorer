Welcome to the COG explorer!

This app allows you to inspect single scenes of the Landsat-8 archive on Amazon S3. Either copy the URL of the index.html of the scene you want to investigate. Alternatively, we have already prepared two scenes: just use the two example buttons.

The selected scene will be displayed on the map, which you can zoom and pan as you like. Additionally, you can adjust how the scene is displayed by clicking on the icon.

In the options menu, you can specify what bands you want to use as the red, green and blue channels (or use one of the already prepared scenarios) and specify what image processing steps shall be applied (currently contrast enhancement and gamma are implemented).

This application uses the geotiff.js library to read the actual raster values from the band files (or the overviews, depending on the maps zoomlevel). Taking advantage of the optimized structure of COGs, only the parts of the file that are actually required will be requested, drastically reducing the overall amount of data transmitted from the archive.

If you want to know more about COGs please refer to their introduction page.
